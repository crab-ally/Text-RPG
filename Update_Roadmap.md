# 직업과 스킬 업데이트 순서
1. 바로
  - 마을에 전직소 추가: 전직하는 곳
  - 스탯 재정비: 마력 스탯 추가 / 마력 스탯은 마법사 전직 시만 활성화 / 마법사 전직 시 공격력 스탯을 마법 스탯으로 치환
  - 전직 시 스탯 증감 구현: 하단 구술
  - 스킬 구현: 상세 스킬 하단 구술 / 스킬은 1턴에 1번만 사용 가능 / 적절한 스킬 쿨타임 적용(고레벨 스킬 위주)
2. 이후
  - 훈련소: 스킬 강화
  - 히든 스킬 획득처

# 전직
- 레벨 10 때 직업 선택
- 마을에 전직소 추가
- 전직 시 스탯 변화
    - 스탯: hp / mp / 공격력 / 방어력 / 회피율 / 치명타 확률
    - 전사: 최대 hp와 방어력 증가, 회피율 감소
    - 마법사: 최대 mp와 마력 증가(공격력 -> 마력 변환), 방어력 감소
    
# 전사

> 특징: 높은 체력과 방어, 맞고 버티는 스타일

| 스킬명 | 형태 | 요구레벨 | 소모 | 설명 | 특이사항 |
|---|---|---|---|---|---|
| 강타 | 액티브 | Lv10 | HP | 강한 물리 피해 | |
| 강인한 신체 | 패시브 | Lv20 | - | 최대 HP 증가 및 상태이상 저항력 증가 | |
| 돌진 기동 | 액티브 | Lv30 | HP | 적에게 빠르게 접근 후 공격 | 낮은 확률 기절 |
| 철벽 자세 | 액티브 | Lv40 | HP | 일정 턴 동안 받는 피해 대폭 감소 & 반격 확률 증가 | |
| 심판의 반격 | 액티브 | Lv50 | HP | 일정 턴 동안 피격 시 반격 | 반격 시 소량의 HP 흡수, 반격 시 적 최대 HP 비례 피해 |
| 투쟁심 | 패시브 | Lv60 | - | HP % 이하: 공격력 증가 & HP % 이하: 치명타 확률 크게 증가 |  |
| 분쇄격 | 액티브 | Lv70 | HP | 방어력 일부 무시 공격 | |
| 광전사의 혼 | 액티브 | Lv80 | HP | 공격력 증가, 회피율 감소, 받는 피해 증가 | 반격 확률 소폭 부여 |
| 강철 심장 | 패시브 | Lv90 | - | HP가 일정 % 이하일 때 턴마다 HP 회복 | |
| 군주 강림 | 액티브 | Lv100 | HP % | 강력한 일격 + 방어력 증가 + 상태이상 면역 | 사용 후 2턴 스킬 사용 불가 (기력 소진)  |


# 마법사

> 특징: 높은 mp와 마력, 딜은 강하나 몸은 약한 하이리스크/하이리턴 스타일

| 스킬명 | 형태 | 요구레벨 | 소모 | 설명 | 특이사항 |
|---|---|---|---|---|---|
| 화염구 | 액티브 | Lv10 | MP | 화염 피해 | |
| 마력 친화 | 패시브 | Lv20 | - | 최대 MP 증가 & 매 턴 MP % 자연 회복 | |
| 빙결의 창 | 액티브 | Lv30 | MP  | 피해 + 둔화 | 적 회피율 감소 |
| 마나 실드 | 액티브 | Lv40 | 유지 중 MP 지속 소모 | 몬스터 공격 피해 흡수 | 최대 MP의 일정 % 한도까지만 흡수 후 자동 해제 (무한 방어 방지) |
| 시간 파열 | 액티브 | Lv50 | MP | 적 1명에게 피해를 주고 시간 균열을 일으킴 | 플레이어 다음 턴 피해 증가 |
| 집중 | 패시브 | Lv60 | - | 마나 70% 이상 시 치명타 확률과 마력 증가 | 마나 소모 증가 (소폭) |
| 폭렬 마법 | 액티브 | Lv70 | MP | 강한 단일 피해 + 화상 | 적 화상 상태이상 |
| 위상 변환 | 액티브 | Lv80 | MP | 적의 공격을 회피하며 마력을 응축 | 1턴 회피 + 치명타 확률 대폭 증가 |
| 현자의 지혜 | 패시브 | Lv90 | - | 스킬 사용 시 확률적으로 MP를 소모하지 않음 | 연속 발동 불가 + 발동 시 다음 스킬 MP 소모 소폭 증가 |
| 메테오 | 액티브 | Lv100 | MP % | 초강력 피해 | 현재 MP가 많을수록 피해 계수 폭증 & MP 회복 불가 (2턴) |

## ⚙️ 구현 팁 (중요)

- 형태 기준 분기:
  - `액티브`: 사용 시 발동 (턴 소비)
  - `패시브`: 항상 적용 (턴 소비 없음)
